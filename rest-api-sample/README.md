# rest-api-sample

## 概要

- Spring Boot 2.5.6
- Java 11
- 以下を使用したサンプル
    - Mybatis Thymeleaf
    - Modaal(未実装)※モーダルウィンドウ
    - lodash(未実装)※配列のソート
    - Pagination.js(未実装)※ページング

## 使い方

アニメ監督の実績を返すAPIの体裁で作成。

IDEで起動して、以下のURLにリクエストして実行。

http://localhost:8080/directorWorks/{directorId}

directorIdはD0000001とD0000002があります。
